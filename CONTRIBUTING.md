# Contributing to ByteSurgeon

Love JVM sorcery? Help us make perf monitoring surgical!

## Quick Start
- Fork → Clone → Branch 
- Install Maven 3.9+.
- PR: Target `main`, describe changes (e.g., "Patches async methods").

## Code Style
- Java 17+.
- Follow Google Java Style.

Issues? Open one. Questions? #bytesurgeon on X @amrxtgh (TBD).
